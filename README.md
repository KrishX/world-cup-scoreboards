# :soccer::trophy: world-cup-scoreboards
A Node.js CLI to show FIFA World Cup 2018 results in real time.

# Usage
Clone this repo and cd to world-cup-scoreboards directory.

Install dependencies.
```
npm install
```

Run the entry file.
```
node index.js
```

Example output:
```
> $ node index.js
  
  Russia 5 x 0 Saudi Arabia
  Egypt 0 x 1 Uruguay
  Morocco 0 x 0 Iran
```
